# A/B Testing Campaign - Neon Genesis

## Goal
Run split A/B tests on the computer vision campaign to compare two visual variants before full launch.

## Variants
- **Variant A (Control)**: Current setup - cyberpunk aesthetic with neon focus
- **Variant B (Treatment)**: High-contrast, dystopian-tech variation with different visual effects

## Steps
1. ✅ Analyzed existing campaign structure
2. Create Variant B storyboard (different prompts & effects)
3. Generate sample preview frames for both variants
4. Create A/B comparison report
5. Document results for launch decision
